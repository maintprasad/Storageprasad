📐 Google Sheets — Struktur Kolom
Buat 2 sheet di Google Spreadsheet dengan nama dan kolom persis seperti ini:
Sheet 1: ITEMS_DB
ID | Item Code | Item Name | Department | Item Type | Moving Type | Area | Purchase Price | Stock | Unit | PIC | Last Update | Notes
Sheet 2: STOCK_LOG
Timestamp | Item Code | Transaction Type | Quantity | Remaining Stock | PIC | Notes
🚀 Cara Deploy (sekali saja oleh admin)
  1. Buka Google Sheet → Extensions → Apps Script
  2. Paste seluruh isi file apps-script-backend.gs
  3. Ganti YOUR_SPREADSHEET_ID_HERE dengan ID spreadsheet
  4. Pilih fungsi setupSheets → klik ▶ Run → grant permission
  5. Deploy → New deployment → Web app
  6. Set Execute as: Me dan Who has access: Anyone
  7. Copy URL /exec
  8. Edit file index.html → ganti nilai API_URL dengan URL tadi
  9. Bagikan file index.html ke seluruh tim maintenance — selesai!
