📐 Google Sheets — Struktur Kolom ITEMS_DB (v3.6.0)
Sheet ITEMS_DB harus memiliki kolom persis seperti berikut (15 kolom):
A: ID  |  B: Item Code  |  C: Item Name  |  D: Brand  |  E: Spec  |  F: Item Type  |  G: Item Class  |  H: Moving Type  |  I: Purchase Price  |  J: Stock  |  K: Min Stock  |  L: UOM  |  M: PIC  |  N: Last Update  |  O: Notes
Sheet 2: STOCK_LOG
Timestamp | Item Code | Transaction Type | Quantity | Remaining Stock | PIC | Notes
Field mapping (GAS → HTML)
GAS kirim: itemType → HTML baca: item.itemType ✓
GAS kirim: itemClass → HTML baca: item.itemClass ✓
GAS kirim: movingType → HTML baca: item.movingType ✓
GAS kirim: minStock (kolom K) → HTML baca: item.minStock ✓
HTML kirim ke GAS addItem/editItem: itemType (field name) ✓
🚀 Cara Deploy (sekali saja)
  1. Buka Google Sheet → Extensions → Apps Script
  2. Paste isi file apps-script-backend-v3.6.0.gs
  3. Ganti SPREADSHEET_ID dengan ID spreadsheet kamu
  4. Jalankan fungsi setupSheets() → grant permission
  5. Jika upgrade dari v3.5.0, jalankan migrateItemsDataShift() untuk tambah kolom K
  6. Deploy → New deployment → Web app (Execute as: Me, Access: Anyone)
  7. Copy URL /exec
  8. Edit API_URL di file HTML ini dengan URL tersebut
